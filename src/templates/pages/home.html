Welcome in MyWallet app!

Introduction

MyWallet is Web application that allows user to manage his stock market wallet. Project is part of the course AiTSI (Architektura i Technologie systemów informatycznych) on OKNO PW studies.

Key functionalities

The key functionalities of the application are:

  • User can input his stock market transactions.
  • User can edit or delete stock market transactions.
  • User can find last known market price of the asset when entering transaction.
  • User can see the return on investment portfolio based on transactions data and current stock values.
  • User can see return per asset.
  • User can see stock price charts.

Stock market data source

Stocks data is taken from historical values available on stooq.pl. Based on this data application computes the return on investment portfolios.

Technical stack

Application is written in python using framework Flask which was presented during the course.

Data is store in SQL database

JavaScript library jQuery is utilized for various client-side operations within the application, enhancing user experience and interactivity.

Why this application fullfil AiTSI (Architektura i Technologie systemów informatycznych) project requirements?

Author thinks that this application fullfil requirements because:

  • Application is using valid HTML which pass all W3C validations.
  • Application contains custom CSS styling. Application does not use Bootstrap library. Only datepicker, chart and icons are imported from external sources.
  • Application is fully functional, it has consistent styling and keep up with current styling trends.
  • Application is functional for disabled people. Navigation with keyboard is possible, images have their alternative texts.
  • Application uses additional technologies like SQL, both JavaScript (for example as an action after button click to fetch current asset price) and server side scripts (to calculate and return current wallet summary)
  • Application has navigation between several tabs - home, wallet details, transactions history, add transaction form, edit transaction form, user profile.
  • Application is deployed and publicly available using Google Cloud services.
